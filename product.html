GCA AI Quant Access Product Spec

This is the public product map for connecting the Go China Access narrative to practical non-custodial quant risk tools. It turns GCA from a token-only story into an account and membership layer for research, risk review, backtests, service credits, and member workflows.

This page remains the public product spec only. Six browser-only risk tools are live public previews: position sizing, risk warning, ENTRY_READY review, Backtest Lab, Liquidation Replay, and Trade Journal. It is not a live trading terminal, not live market data, not financial advice, and does not enable order execution.

Safety Architecture

Custody No custody

The product path must not hold user funds and must keep no withdrawal permission.

Secrets No private keys

No seed phrases, private keys, withdrawal-enabled API keys, or exchange API secret collection.

Orders No live execution now

Any future order flow must use unique client order IDs and pass risk checks first.

Validation Simulation first

Trading-related features must run simulated or testnet validation before live use.

Safe To Say

  • GCA has published a public product specification for GCA AI Quant Access.
  • GCA has a live account and membership intake path for non-custodial quant risk tools.
  • Any future trading-related feature must stay behind risk checks, simulation or testnet validation, and controlled account UI.
  • The official public market route is the Base Mainnet Uniswap v4 GCA/USDT pool.
