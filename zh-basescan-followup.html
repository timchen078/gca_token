收到 BaseScan 邮件后先做三件事

1. 记录 ticket保存 ticket id、邮件时间、发件人、主题和 BaseScan 原文；不要只看机器翻译。
2. 判断类型区分是已经更新、继续退回、要求补资料、要求私密证据，还是只是服务满意度调查。
3. 不重复提交同一个问题先回复同一个 ticket；只有公开资料修完后，才考虑新的 clean resubmission。

如果 BaseScan 说已经处理

先打开 BaseScan token page 验证公开资料是否真的显示 Logo、官网、邮箱、社群和说明。只有公开页面显示后，才能说 BaseScan Token Profile 已发布。

如果仍说信息不足

先不要重新提交。对照本页“退回原因映射”，修官网和证据链接，再用 中文复审复制包 准备下一次干净回复。

如果要求私密证据

只通过同一个 BaseScan ticket 补充 reviewer 明确要求的截图或文件；邮箱后台、DNS 后台、登录截图不要公开放到官网。
