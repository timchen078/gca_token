参与顺序

按这个顺序做

不要从陌生链接、私聊链接或截图二维码开始。所有操作先回到官网验证页。

1. 核对链钱包必须切到 Base Mainnet / chainId 8453。
2. 核对合约GCA 合约必须是 0x3197c42f4a06f7be32a9a742ac2a766f0ff682c6。
3. 核对交易路线官网公开路线是 Base 上的 GCA/USDT 池，不要使用旧截图或陌生池子链接。
4. 小额测试如果你决定交易，先小额测试买入和卖出，确认钱包显示、滑点和成交结果。
5. 准备会员资料如果要进入会员路径，保存钱包地址、购买交易哈希、持有开始日期和只读余额验证结果。
6. 提交账户入口账户资料、只读钱包验证和符合条件账本记录可在 gca/member-access/ 提交；会员权益转账仍不能说成自动领取。

普通持有者路径

如果你只是想关注项目，先看 验证页、中文 FAQ、中文池子和流动性说明、中文总量和储备说明、中文安全和审计说明 和 官方市场页。钱包不显示 GCA 时，切到 Base Mainnet 后用合约地址手动导入。

10,000 GCA 路径

计划规则：每个注册用户只能审核一个钱包。只读余额验证显示至少 10,000 GCA 后，可准备未来 100 GCA AI Quant Access credits 审核。当前不是公开自助领取。

1,000,000 GCA Member 路径

计划规则：购买并连续持有至少 1,000,000 GCA 满 30 天，才可进入 GCA Member 审核。GCA Member 仍需要支持审核和账本激活。

10,000 GCA 会员权益审核

通过 GCA Member 审核后，才可能进入一次性 10,000 GCA 会员权益审核。它不是现金、收入、交易结果承诺、费用返还、自动发放或风控豁免。

你应该保留什么

  • Base Mainnet 钱包地址。
  • 购买或转入 GCA 的公开交易哈希。
  • 持有开始日期和后续 30 天的余额复核记录。
  • 官方支持沟通记录和会员确认状态。
  • 只读余额验证截图或导出的本地预登记包。

现在还不能承诺什么

  • 不能承诺 BaseScan 公开代币资料已经通过；目前仍以 BaseScan 最终公开页面为准。
  • 不能承诺第三方审计已完成；目前没有独立审计报告。
  • 不能承诺 LP 已锁、储备已锁仓或储备已多签。
  • 不能承诺会员权益自动发放或一定通过审核。
  • 不能承诺价格、流动性、交易量、上币或钱包永久无提示。

安全边界

官方不会索要这些信息

任何参与路径都只需要公开链上资料和只读验证。GCA 官方不会索要私钥、助记词、交易所 API Secret、提现权限、验证码、恢复短语、资金托管或代付请求。

  • 不要向任何人发送私钥、助记词或验证码。
  • 不要给任何交易所 API 开通提现权限。
  • 不要相信要求额外担保费、手续费返还或私下打款的领取流程。
  • 不要把本地预登记包当成链上交易或正式会员批准。
  • 不要通过人工制造交易活动或误导性宣传来改善市场数据。
