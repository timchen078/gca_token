怎么买 GCA？

先打开 验证页 确认合约和链，再看 中文购买说明。官方公开路线是 Base Mainnet 上的 GCA/USDT 池。交易前请确认钱包在 Base Mainnet，使用的是 Base USDT，并注意滑点和价格影响。

为什么钱包里看不到 GCA？

钱包不显示通常不是余额消失。请在 MetaMask 里切到 Base Mainnet，然后用 GCA 合约地址手动导入代币。价格显示也可能延迟，因为钱包需要数据源识别池子和代币资料。

为什么有些平台显示流通量是 10 亿？

ERC-20 合约在部署时一次性创建 1,000,000,000 GCA。数据平台如果只读取总量，可能会把全部供应视为可流通。GCA 已公开说明 600,000,000 GCA 在 owner-held reserve 地址，但它目前不是锁仓、多签或归属合约，所以平台可能不会自动扣除。

别人买了 GCA，我的钱包会收到钱吗？

如果别人从 DEX 池子买，资金进入池子并改变池子里的 USDT / GCA 储备，不会自动进入项目方个人钱包。只有提供流动性的地址在管理 LP 头寸、收取费用或移除流动性时，才会产生对应的 LP 层面变化。

别人买币，我钱包里的 GCA 会变少吗？

不会。别人从池子买，减少的是池子里的 GCA，不会动你的钱包。你的 GCA 只会在你自己转账、卖出、添加流动性或签名确认相关交易时减少。

如果不走池子，怎么结算？

不走池子的直接转账或场外成交，需要买卖双方自行约定付款和转币。官网不提供托管、收款担保或代收代付流程。为了公开透明，普通用户应优先使用官网公开的 DEX 路线和链上可查记录。

现在有没有第三方审计？

没有。GCA 已有内部技术 review 和公开技术报告，但这不能写成第三方审计。只有独立审计公司发布报告后，才能公开说外部审计完成。

风险提示消失是不是永久安全？

不是。钱包和 DEX 风险提示来自第三方系统，可能因为流动性、持仓集中度、价格波动、资料完整度或模型规则变化而出现或消失。官网只能公开合约事实、审核资料和当前状态，不能保证所有钱包永久无提示。

会员规则是什么？

10,000 GCA 持有者可通过账户入口提交 100 Web3 Radar utility credits 账本审核；购买并连续持有 1,000,000 GCA 满 30 天，可进入 GCA Member 账本审核和 10,000 GCA 会员权益人工审核。credits 和 GCA Member 账本记录可以写入，但所有权益都不能说成现金、收入或自动转账。

现在最重要的下一步是什么？

继续保持公开资料一致，等待 BaseScan token profile 审核结果，增加真实社区内容，完善受控 HTTPS 账户 UI、只读钱包验证、会员账本和支持审核流程。市场质量只能通过真实信息、真实用户和更透明的流动性建设提升。

安全边界

任何时候都不要发送这些信息

GCA 官方流程不会索要你的私钥、助记词、交易所 API Secret、提现权限、验证码、恢复短语、托管请求或资金转移请求。

  • 不要把 GCA 说成已经通过 BaseScan 公开代币资料审核；目前是 2026-05-23 被退回整改，Tim Chen 公开职业资料已发布，仍需补域名邮箱后再提交。
  • 不要把内部技术 review 说成外部审计；第三方审计尚未完成。
  • 不要把 owner-held reserve 说成锁仓、多签或归属合约，除非未来链上真的完成对应动作。
  • 不要把会员权益说成现金、收入、交易权限、自动领取或风险控制豁免。
  • 不要把小额测试交易说成真实市场需求或深度流动性证明。
  • 不要通过人工制造交易量、自我成交或误导性宣传来改善市场数据。
