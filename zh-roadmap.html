公开宣传边界

现在可以说什么，不能说什么

GCA 可以讲清楚愿景和路线图，但不能把计划功能说成已交付，也不能把会员权益说成现金或交易结果。

可以说

  • GCA 是 Base Mainnet 上固定总量 ERC-20，合约没有后续增发函数。
  • GCA 正在建设 Go China AI Quant Access 方向的会员访问和研究工具入口。
  • 官方交易路线是 Base Mainnet 上的 GCA/USDT 池。
  • 10,000 GCA 和 1,000,000 GCA / 30 天规则是公开的会员审核规则。

不能说

  • 不能说 10,000 GCA 会员权益公开自助领取已经上线。
  • 不能说第三方审计已经完成。
  • 不能说 BaseScan Token Profile 已经通过，除非 BaseScan 页面公开显示。
  • 不能承诺价格、成交量、流动性、上所、审核通过或交易结果。
