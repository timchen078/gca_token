This journal stores data in browser localStorage only. It does not upload trades, connect accounts, fetch market data, place orders, or deduct credits. A completed Trade Plan can prefill bounded market, direction, setup, and note context, but never the date or realized return and never saves a journal entry automatically. Do not enter private keys, passwords, API secrets, one-time codes, or withdrawal permissions. Clearing browser storage can remove the journal, so export a backup when needed.
