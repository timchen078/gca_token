GCA 中文数据室说明

JSON 文件不是坏页面，也不是领取入口。它们是 Raw JSON / machine-readable files，看起来像代码是正常的，主要给钱包、浏览器、DEX 数据工具、上币审核、数据平台和安全厂商使用。

普通用户优先 HTML / 平台审核才用 JSON。买币、注册、会员资料、钱包验证、支持沟通和项目介绍，都应该先打开中文 HTML 页面。

JSON 文件不是坏页面

如果你点开后看到大括号、字段名、链接和代码样式，这是正常显示。它不是网站故障，也不是浏览器中毒。

JSON 不是领取入口

JSON 不会发币、不会创建账户、不会激活 100 credits、不会激活 GCA Member、不会发放 10,000 GCA 会员权益。

平台审核才使用

只有 BaseScan、钱包、DEX 数据平台、上币审核或安全厂商明确要求结构化资料时，才需要打开 Reviewer Data Room。

安全边界

打开任何资料都不需要这些信息

钱包资料不需要私钥、助记词、keystore 文件、钱包密码或陌生签名
账户资料不需要验证码、交易所登录密码、交易所 API Secret 或提现权限
设备权限不需要远程控制权限，不需要任何人接管电脑、手机、浏览器或钱包插件
链上动作数据室不会要求 approve、授权、转账或支付费用
